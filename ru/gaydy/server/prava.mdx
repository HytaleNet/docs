## Команды администратора

Эти команды предназначены для управления игроками, плагинами и состоянием сервера.

| **Команда**  | **Описание**                                                                 |
| :----------- | :--------------------------------------------------------------------------- |
| `/op`        | Назначить игрока (или себя) администратором сервера.                         |
| `/perm`      | Основная команда для управления правами (permissions) групп и пользователей. |
| `/spawning`  | Группа команд для управления спавном NPC.                                    |
| `/ban`       | Заблокировать игрока на сервере (бан).                                       |
| `/unban`     | Разблокировать ранее забаненного игрока.                                     |
| `/kick`      | Принудительно кикнуть игрока с сервера.                                      |
| `/whitelist` | Управление «белым списком» (допуск только определенных игроков).             |
| `/gamemode`  | Изменить режим игры (Выживание, Креатив и т.д.).                             |
| `/give`      | Выдать определенный предмет игроку.                                          |
| `/heal`      | Полностью восстановить здоровье и выносливость.                              |
| `/plugin`    | Основная команда для управления установленными плагинами.                    |
| `/sudo`      | Выполнить команду от имени другого игрока.                                   |
| `/tp`        | Телепортация к игроку или по заданным координатам.                           |
| `/stop`      | Принудительное и безопасное выключение сервера.                              |

## Управление миром

Используйте эти команды для изменения окружения, игровых правил и погодных условий.

| **Команда** | **Описание**                                                  |
| :---------- | :------------------------------------------------------------ |
| `/world`    | Главная команда для управления всеми мирами на сервере.       |
| `/time`     | Изменить время суток в игровом мире.                          |
| `/weather`  | Изменить погодные условия (ясно, дождь и т.д.).               |
| `/lighting` | Управление и получение информации об освещении в мире.        |
| `/block`    | Команда для работы с состояниями блоков и отладки.            |
| `/chunk`    | Управление чанками (информация, принудительная прогрузка).    |
| `/fluid`    | Управление жидкостями в определенном радиусе или координатах. |
| `/path`     | Настройка путей патрулирования для NPC.                       |

## Команды строителя (Builder Tools)

Инструменты для быстрого возведения структур, копирования областей и работы с префабами.

| **Команда**       | **Описание**                                                   |
| :---------------- | :------------------------------------------------------------- |
| `/pos1` / `/pos2` | Установить первую и вторую точки выделения области.            |
| `/copy`           | Скопировать выделенную область в буфер обмена.                 |
| `/cut`            | Вырезать (удалить) содержимое выделенной области.              |
| `/paste`          | Вставить скопированный объект из буфера обмена.                |
| `/undo`           | Отменить последнее действие (откат изменений).                 |
| `/fillblocks`     | Заполнить пустоту в выделенной области указанным типом блока.  |
| `/replace`        | Заменить один тип блоков на другой в выделенной области.       |
| `/prefab`         | Сохранить, загрузить, удалить или просмотреть список префабов. |
| `/editprefab`     | Основная команда для создания или редактирования префабов.     |

## Команды игрока

Общие команды для взаимодействия, проверки статистики и передвижения.

| **Команда** | **Описание**                                                    |
| :---------- | :-------------------------------------------------------------- |
| `/help`     | Вывести в чат список всех доступных вам команд.                 |
| `/whoami`   | Показать подробную информацию о вашем игроке в чате.            |
| `/ping`     | Показать задержку (пинг) игрока и серверные тики.               |
| `/spawn`    | Телепортироваться на точку спавна мира.                         |
| `/warp`     | Управление точками телепортации (варпами) и перемещение по ним. |
| `/kill`     | Убить выбранного игрока (самоубийство, если цель не указана).   |
| `/damage`   | Нанести указанный урон выбранному игроку.                       |
| `/mount`    | Управление ездовыми животными (спешиться, проверить статус).    |
| `/emote`    | Воспроизвести анимацию эмоции персонажа.                        |
